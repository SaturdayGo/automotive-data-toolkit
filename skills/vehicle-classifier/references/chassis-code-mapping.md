## 使用说明

1. **查询映射**：使用 `scripts/classify.py --lookup <底盘代号>` 查询单个映射
2. **验证映射**：使用 `scripts/verify_mapping.py` 验证映射数据完整性
3. **更新映射**：编辑 `assets/mapping-data.json`，然后运行验证脚本

## 维护指南

添加新映射时：
1. 先在 Wikipedia 查找对应车型页面
2. 确认底盘代号与市场名称的对应关系
3. 记录年份范围
4. 更新 `assets/mapping-data.json`
5. 运行 `scripts/verify_mapping.py` 验证
